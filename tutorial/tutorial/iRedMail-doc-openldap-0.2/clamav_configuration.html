8.2. 配置 ClamAV

clamd 的配置文件是 /etc/clamd.conf，基本上不需要配置，唯一可能要留意的是，你是否需要让 clamd 监听来自其它主机的请求： 

#
# File: /etc/clamd.conf
#

TCPAddr 127.0.0.1

如果不需要监听来自其它主机的请求，就只使用本地 Unix Socket 的方式，另外将参数 'TCPAddr' 设置为 'no'： 

TCPAddr no

clamd 默认的日志文件是： 

/var/log/clamav/clamd.log
