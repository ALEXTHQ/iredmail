7.1. 安装 ClamAV

安装 ClamAV： 

# yum install clamav clamav-db clamd

如果需要用 clamav 扫描 RAR/ARJ 压缩文件，还需要安装 unrar, unarj： 

# yum install unrar unarj

启用 clamav： 

# chkconfig --level 35 clamd
