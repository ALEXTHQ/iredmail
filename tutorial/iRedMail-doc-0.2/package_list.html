2.3. 使用的软件列表

本邮件服务器基于 OpenBSD 4.2 -release，i386 平台搭建，主要使用到了以下软件：

  • Postfix

  • OpenLDAP

  • Berkeley DB

  • Courier IMAP/POP3/Maildrop, authlib

  • Cyrus-SASL

  • PHP5

  • phpLDAPadmin

详细的软件包名称及其版本号可以查看文件：package list。

软件包

以上软件包都可以在 OpenBSD 官方站点下载到，并且使用 binary packages 直接安装：

  • ftp://ftp.openbsd.org/pub/OpenBSD/4.2/packages/i386/

镜像站点

您可以从离自己较近的镜像站点下载。镜像站点的列表可以在 http://www.openbsd.org/ftp.html 页面找到。 中国的用户可以使用以下几个镜像站点：

  • TaiWan：ftp://ftp.tw.openbsd.org/pub/OpenBSD/4.2/packages/i386/

  • Japan：ftp://ftp.jp.openbsd.org/pub/OpenBSD/4.2/packages/i386/

Postfix

由于官方打包的 binary packages 里的 postfix 不能同时支持 SASL 和 LDAP，所以 postfix 必须使用 ports 编译。
