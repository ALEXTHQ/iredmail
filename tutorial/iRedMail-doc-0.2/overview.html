2.1. 功能列表

  • 虚拟域、虚拟用户

    • 使用非 rootdn 作为邮件服务器虚拟域和虚拟用户的管理员，权限最小化。

    • 管理员可以使用 vmail/vmailadmin 分别查看和管理所有虚拟域和用户。管理操作包括域、用户的添加、删除、修改等。

    • 独立的域管理员。每个域有一个单独的管理员账号，用于本域内的账号管理。如用户的添加、删除、修改等。

  • SMTP

  • POP3/IMAP，SSL support

  • 邮箱容量限额

  • WebMail

    • 使用 SquirrelMail 作为默认的 WebMail。

    • 完善的多语言、国际化（i18n）支持。默认使用 utf-8 字符集。

    • 提供全局地址簿功能。用户可以看到本域内的所有用户。

    • 提供日历功能。

  • 邮件备份

  • 病毒防护

  • 垃圾邮件过滤
