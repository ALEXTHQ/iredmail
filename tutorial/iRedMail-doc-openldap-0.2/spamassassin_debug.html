9.5. 调试 SpamAssassin

要调试 SpamAssassin，只需要在 /etc/amavisd.conf 中打开这两个参数，然后重启 amavisd 服务即可： 

$log_level = 5;
$sa_debug = 1;

注意：只有在设置 $log_level 之后，'$sa_debug = 1;' 才会生效。

重启 Amavisd 服务： 

# /etc/init.d/amavisd restart

Caution

Always run 'spamassassin -D --lint' before making your rule changes live. Typos will kick your butt! :)
